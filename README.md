## Description
**xyloPhone** is a repo for a simple swiftUI app that turns your iOS device into a pretty xylophone. 

## Skills Practiced

* Gradients
* AVFoundation
* ObservableObjects
* Geometrey Readers
* Custom Button Style

## Future Features / Fixes
* Add launch screen
* Record music played?

## Use Instructions

This simple app should work immediately after cloning. Simply run in simulator.
